# 🎬 Netflix Data Analytics Project

## 📌 Overview

This project focuses on analyzing Netflix-style streaming data to extract meaningful insights using **SQL** and **Power BI**. The goal is to transform raw data into actionable information that helps understand content trends, user preferences, and platform performance.

## 📊 Project Workflow

### 1️⃣ Data Collection

* Imported dataset containing Netflix content details
* Data includes: Title, Genre, Release Year, Ratings, Duration, Country, etc.

### 2️⃣ Data Cleaning (SQL)

* Removed null and duplicate values
* Standardized inconsistent formats (dates, text fields)
* Handled missing values using SQL queries

### 3️⃣ Data Transformation (SQL)

* Created meaningful columns (e.g., content categories, duration groups)
* Aggregated data for analysis (genre-wise, country-wise, yearly trends)
* Used joins, group by, and window functions for deeper insights

### 4️⃣ Data Analysis

Key questions answered:

* Which genres are most popular?
* Which countries produce the most content?
* How has content production changed over time?
* What is the distribution of Movies vs TV Shows?

### 5️⃣ Data Visualization (Power BI)

* Built interactive dashboards
* Used filters and slicers for dynamic exploration
* Created charts such as:

  * Genre distribution
  * Yearly content growth
  * Country-wise content production
  * Ratings analysis
